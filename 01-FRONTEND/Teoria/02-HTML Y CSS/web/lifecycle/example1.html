Ejemplo de DOMContentLoaded

El evento DOMContentLoaded se dispara cuando el HTML se ha cargado y el DOM está completamente construido, pero no necesariamente los recursos externos como imágenes o hojas de estilo. Este evento es útil para manipular el DOM en cuanto está listo. • El evento DOMContentLoaded es capturado usando addEventListener. En este punto, el DOM está listo, pero la imagen puede no haberse cargado completamente (por lo que los tamaños pueden ser incorrectos). • Utilidad: Iniciar lógica que dependa de la estructura HTML sin esperar a la carga completa de los recursos externos.
