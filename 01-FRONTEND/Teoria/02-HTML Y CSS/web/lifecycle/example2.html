Ejemplo de evento Load

El evento load se dispara cuando todos los recursos de la página (imágenes, estilos, etc.) se han cargado completamente. Este es el mejor momento para hacer algo que dependa de recursos externos. • load espera a que la imagen y todos los recursos se carguen antes de ejecutarse. • Utilidad: Trabajar con imágenes, fuentes, o estilos que dependen de ser completamente cargados.
