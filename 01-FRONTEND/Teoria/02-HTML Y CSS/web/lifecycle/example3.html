Ejemplo de evento Beforeunload

Los eventos beforeunload y unload se disparan cuando el usuario está a punto de abandonar la página o cerrar la ventana. beforeunload permite pedir confirmación al usuario antes de salir. • beforeunload permite mostrar una advertencia antes de que el usuario cierre la página. • Utilidad: Confirmar con el usuario si realmente desea salir si tiene cambios sin guardar.
